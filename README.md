# Wavy Pixelated Flag
Wavy pixellated flags with neat react and css tricks
## Features
- Click & Drag around the Flag
- Flag Oscillation
- Flag Picker (horizontal stripe flags for now)
## Demo
- GUI for Flag variables control
- Draggable Flag
## Tech Stack
- React
- lil GUI
